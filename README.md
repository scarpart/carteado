
# carteado - An Emacs spaced repetition mode

This script performs spaced repetition inside Emacs for better memorization. It can work as a frontend client for AnkiConnect, or as a standalone application running its own implementation of FSRS. Collections are directories and cards are Org files following the templates inside them. Writing to a file creates the card in the Anki collection automatically and it is included in the next study iteration.
